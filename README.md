AI-Based Sentiment Analyzer

Description

An AI-powered web application built using Python and Flask to analyze product reviews and social media comments.

Features

- Analyze text sentiment
- Predict Positive, Neutral or Negative sentiment
- Display confidence percentage
- Show visual charts

Technologies Used

- Python
- Flask
- VADER Sentiment Analysis
- Chart.js
